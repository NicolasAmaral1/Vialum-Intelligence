# Story 2.5.1: Vialum Chat — Media persist + textContent

## Status: Ready

## Executor
executor: "@dev"
quality_gate: "@qa"

## Story
**As a** operador ou IA do Vialum Chat,
**I want** que toda midia recebida via WhatsApp seja persistida automaticamente
no Media Service e que o conteudo (texto transcrito, OCR) seja salvo na mensagem,
**so that** a IA consiga ler o contexto completo de uma conversa incluindo audios,
imagens e documentos como texto.

## Acceptance Criteria
1. Quando mensagem com midia chega (image, audio, document, video), o Vialum Chat
   enfileira job `media-persist` que chama Media Service /files/from-whatsapp
2. O `contentAttributes` da mensagem é atualizado com `mediaFileId` (referencia permanente)
3. Media Service recebe o arquivo → persiste no MinIO → emite webhook file.created
4. Switch recebe webhook → auto-processa → emite webhook job.completed
5. Vialum Chat recebe webhook job.completed e salva `text_content` na mensagem:
   - Audio: "[Audio] texto transcrito..."
   - Imagem: "[Imagem] texto extraido via OCR..."
   - Documento: "[Documento] texto extraido..."
6. GET /conversations/:id/messages retorna `textContent` preenchido
7. Campo `text_content` adicionado na tabela messages (migration)
