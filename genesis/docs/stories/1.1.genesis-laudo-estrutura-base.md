## Status: Done

## Executor

## Story

**As a** desenvolvedor do projeto Vialum,
**I want** um serviço Node.js com Express/EJS estruturado e dockerizado,
**so that** tenha uma base sólida para construir o dashboard HITL do genesis-laudo.

## Acceptance Criteria

1. Diretório `genesis-laudo/` criado com estrutura `app/routes`, `app/views`, `app/public`, `app/services`, `app/migrations`
2. `package.json` com dependências: express, ejs, pg, marked
3. `server.js` com Express + EJS, PORT 3001, startup async com `waitForDb`
4. `GET /laudo/health` retorna `{"status":"ok","service":"genesis-laudo"}`
5. `GET /laudo/` retorna HTTP 200 (dashboard skeleton)
6. `Dockerfile` Node.js 20 Alpine funcional
7. `genesis-laudo.yml` Docker Compose com App + Postgres dedicado + Traefik labels
8. `.env.example` com as variáveis necessárias
