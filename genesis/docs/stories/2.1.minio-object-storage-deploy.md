## Story
**As a** operador do ecossistema Vialum,
**I want** um object storage S3-compatible (MinIO) rodando na VPS,
**so that** todos os servicos (Media Service, Classification Hub, squads) tenham
um ponto central, auditavel e escalavel para persistir e servir arquivos.

## Acceptance Criteria
1. MinIO rodando como container Docker na VPS (vps-nova)
2. Conectado a rede `proxy` do Traefik
3. API S3 acessivel via `https://s3.luminai.ia.br`
4. Console admin acessivel via `https://s3-console.luminai.ia.br`
5. Certificado TLS via Let's Encrypt (certresolver=le)
6. Buckets criados: `vialum-media`, `vialum-classifications`, `vialum-portal`, `vialum-backups`
7. Todos os buckets com policy `none` (privados)
8. Lifecycle rule no bucket `vialum-classifications`: expirar apos 90 dias
9. Volume persistente `minio-data` para dados
10. Health check configurado
11. Memory limit: 256MB
12. Credenciais seguras (nao default)
