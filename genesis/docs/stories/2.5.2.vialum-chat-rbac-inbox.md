# Story 2.5.2: Vialum Chat — RBAC por Inbox

## Status: Done

## Executor
executor: "@dev"
quality_gate: "@qa"

## Story
**As a** admin do Vialum Chat,
**I want** que cada usuário só veja as conversas das inboxes atribuídas a ele,
**so that** vendedores vejam apenas suas próprias conversas e admins tenham visão geral do time.

## Acceptance Criteria
1. [x] Tabela `InboxMember` criada — vincula AccountUser a Inbox(es)
2. [x] Campo `supervisorId` adicionado ao AccountUser — quem supervisiona quem
3. [x] Roles expandidos: owner, admin, supervisor, agent
4. [x] Admin/owner veem todas as inboxes
5. [x] Supervisor vê suas inboxes + dos supervisionados
6. [x] Agent vê apenas inboxes atribuídas via InboxMember
7. [x] GET /inboxes retorna flag `isMine` e `canSeeAll` no meta
8. [x] GET /conversations filtra por inboxes acessíveis (RLS) baseado no userId do JWT
9. [x] WebSocket `subscribe:conversation` verifica inbox access (não só accountId)
10. [x] Frontend: Inbox switcher com seções "Minhas inboxes" e "Inboxes do time"
11. [x] Seleção de inbox persistida no localStorage
12. [x] Auto-seleciona inbox do usuário no primeiro acesso
