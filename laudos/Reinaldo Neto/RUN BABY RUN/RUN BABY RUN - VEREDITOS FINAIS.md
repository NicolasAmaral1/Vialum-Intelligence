**Justificativa:** A estrutura sintática é idêntica (RUN [X] RUN) e a sobreposição de specs na classe 41 é alta (competições desportivas, instalações esportivas). **Estratégias diferenciativas:**

1. **Argumento de estrutura aberta:** A construção "RUN [X] RUN" é um padrão imperativo genérico na língua inglesa (como "eat, pray, love"). Não é apropriável. Prova: ambas coexistem com INFINITY RUN STREET RUN e centenas de outras marcas RUN na cl.41.

2. **Argumento do núcleo distintivo:** O INPI avalia o "conjunto marcário" (Manual 5.11). O elemento central é o diferenciador: GEEK remete a cultura nerd/tecnológica, BABY remete a linguagem motivacional esportiva. São identidades de marca completamente distintas. Um consumidor de "run GEEK run" busca eventos de corrida para o público geek; um consumidor de "RUN BABY RUN" busca comunidade wellness e corrida.

3. **Argumento de desgaste:** RUN é genérico na cl.41 (167 marcas). A moldura RUN...RUN é apenas duplicação de elemento genérico. O que sobra como distintivo é GEEK vs BABY — que são NULA em similaridade fonética, gráfica e ideológica entre si.

4. **Precedente favorável:** A própria run GEEK run está em vigor coexistindo com todas as outras marcas RUN na cl.41. Se a estrutura RUN [X] RUN fosse protegível, teria sido indeferida. Não foi.

5. **Diferenciação visual:** Registrar RUN BABY RUN como marca mista com identidade visual própria (tipografia, logo) que não se confunda com a apresentação de run GEEK run.
